⚠️ NEB Theory & Practical Pass Rule
A student must pass Theory and Practical SEPARATELY:
• Minimum 35% in Theory marks (out of thMax)
• Minimum 35% in Practical marks (out of prMax, if subject has practical)
• If either is below 35%, the entire subject is graded NG regardless of total.
• CGPA = Sum of all subject GPAs ÷ Number of subjects
• A student with any NG subject is considered Failed.

⚠️ Saving on Mobile vs Laptop?
Marks are saved in your browser's local storage — not on a server. This means:
• Marks saved on your phone will NOT automatically appear on your laptop (different browser = different storage).
• Marks saved by teacher login ARE visible when admin logs in on the same device/browser.

✅ To share marks across devices: Click "Export & Share File" below → send the file via WhatsApp/Email → open on the other device.
📦 How sharing works:
1. Enter & save all marks here.
2. Click "Export & Share File" — a .html file downloads.
3. Send that file via WhatsApp / Email / USB to anyone.
4. They open the file and see all marks already loaded.
5. They can log in, add more marks, and share back the same way.
⚠️ You opened a shared file. All marks from the sender are loaded. Log in to add more marks, then re-export to share back.

📲 Cross-Device Sync — Share Marks Across Phone & Laptop
⚠️ Why marks are missing on laptop: Marks are saved in your browser's local storage — each device/browser has its own separate storage. Use any method below to sync data between devices.
📁 Method 1: Export File (WhatsApp / Email / USB)

Downloads an HTML file with ALL marks embedded. Open it on any device — no server needed.

Steps: Click Export → Send file via WhatsApp/Email/USB → Open file on other device → Log in → All marks appear ✅
📷 Method 2: QR Code — Scan from Phone

Generate a QR code from laptop → scan with phone camera → marks load instantly on phone.

📱 On your phone: Open camera app → Point at QR code → Tap the link → Marks will load automatically.

Copy the data code below → paste it on the other device in the "Import" box below → marks load instantly.

☁️ Method 5: Cloud Sync — Real-Time (Any Device, Any Time)
🌐 One-click cloud sync! Push all marks to cloud → instantly available on any device, any browser — no file sharing needed. Uses your school's private Firebase database.
Status: Not connected
How to use:
• Push — Upload all marks from this device to cloud.
• Pull — Download latest marks from cloud to this device.
• Live Sync — Auto-update this device when any other device saves marks.
• All devices must use the same Room ID.
⚠️ Setup required: You need a free Firebase project. Show setup guide ▼
📋 Firebase Setup (one-time, 5 minutes):
1. Go to console.firebase.google.com
2. Create a new project (free Spark plan is enough)
3. Go to Realtime Database → Create database → Start in test mode
4. Copy your database URL (e.g. https://your-project-default-rtdb.firebaseio.com)
5. Go to Project Settings → Your apps → Add web app → Copy the firebaseConfig object
6. In this file, find the comment // FIREBASE CONFIG — REPLACE THIS and paste your config

Current config status: ⚠️ Using demo config — replace with your own
