⚠️ Saving on Mobile vs Laptop?
Marks are saved in your browser's local storage — not on a server. This means:
• Marks saved on your phone will NOT automatically appear on your laptop (different browser = different storage).
• Marks saved by teacher login ARE visible when admin logs in on the same device/browser.

✅ To share marks across devices: Click "Export & Share File" below → send the file via WhatsApp/Email → open on the other device.
📦 How sharing works:
1. Enter & save all marks here.
2. Click "Export & Share File" — a .html file downloads.
3. Send that file via WhatsApp / Email / USB to anyone.
4. They open the file and see all marks already loaded.
5. They can log in, add more marks, and share back the same way.
⚠️ You opened a shared file. All marks from the sender are loaded. Log in to add more marks, then re-export to share back.

📲 Cross-Device Sync — Share Marks Across Phone & Laptop
⚠️ Why marks are missing on laptop: Marks are saved in your browser's local storage — each device/browser has its own separate storage. Use any method below to sync data between devices.
📁 Method 1: Export File (WhatsApp / Email / USB)

Downloads an HTML file with ALL marks embedded. Open it on any device — no server needed.

Steps: Click Export → Send file via WhatsApp/Email/USB → Open file on other device → Log in → All marks appear ✅
📷 Method 2: QR Code — Scan from Phone

Generate a QR code from laptop → scan with phone camera → marks load instantly on phone.

📱 On your phone: Open a QR scanner app (or Google Lens) → Scan the code → Copy the full text that appears → Open this app on the phone → Go to Sync Devices → Method 3 Import → Paste the text → Tap Import & Load Marks ✅
📋 Method 3: Copy-Paste Data Code

Copy the data code below → paste it on the other device in the "Import" box below → marks load instantly.

📥 Import Data Code (on the other device)

🔗 Method 4: Shareable Link (if hosted on a server)

If this app is served from a web URL, generate a link with all data embedded. Share via WhatsApp or SMS — tap to open and marks auto-load.

⚙️ Settings: Auto-Remind to Sync After Saving

When ON, you'll get a prompt to export the file every time you save marks — so you never forget to sync.
