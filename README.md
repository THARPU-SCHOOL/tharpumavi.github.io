# Tharpu Secondary School — Result Management System

A Progressive Web App (PWA) for managing and viewing student results for Class XI.

**Live site:** https://tharpu-school.github.io/tharpumavi.github.io/

---

## Features

- Login and Register system with localStorage
- Student result entry and management
- Class XI · Academic Year 2082/83
- Works offline (service worker cached)
- Installable as an app on Android, iOS, and Windows

## Files

| File | Purpose |
|---|---|
| `index.htm` | Main app — login, register, and result management |
| `manifest.json` | PWA metadata (name, icons, theme color) |
| `sw.js` | Service worker for offline support and caching |
| `icon.png` | App icon
